
# ✏️ Style Case Converter

Quickly convert text case (`camelCase`, `snake_case`, etc.).

Install the extension from the **[Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=lonberg.style-case-context-menu)**, select text, right-click, and choose a command from the `✎ Style Case` submenu:

```
✎ Style Case:

    camelCase
    PascalCase
    snake_case
    UPPER_SNAKE_CASE
    ---
    dot.case
    kebab-case
    UPPER.DOT.CASE
    UPPER-KEBAB-CASE
    ---
    Title Case
    Sentence case
```
